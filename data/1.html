1. Для корректного формирования отчета «Боевая работа» и единообразного подхода к заполнению информации по выездам подразделений Гомельское областное УМЧС предлагает:

  2. 1.

    Вкладка «Обработка выезда» - «12 пожар» - «Вид работ» - исключить «в жилом секторе», «пожар памятников архитектуры…», «тушение пожара».


  3. 2.

    Вид работ «на объекте» устанавливать для пожаров, происшедших:
    • в жилых зданиях (одноквартирные, многоквартирные, блокированные жилые дома, общежития, спальные корпуса школ-интернатов, детских домов, дома для престарелых и инвалидов);
    • в нежилых зданиях (сельскохозяйственного, лесохозяйственного назначения, для рыбоводства, горнодобывающей, обрабатывающей промышленности, специализированных зданий транспорта, связи, энергетики);
    • в нежилых зданиях иного назначения (для ремонта и технического обслуживания автомобилей (в том числе автомобильные заправочные и газонаполнительные станции), организаций оптовой торговли, материально-технического снабжения и сбыта продукции, розничной торговли, гостиниц, мотелей, кемпингов, здравоохранения и предоставления социальных услуг, лечебно-санитарного назначения (морги), для общественного питания, складов, торговых баз, баз материально-технического снабжения, хранилищ, научно-исследовательских, проектных, конструкторских организаций, информационных центров, общественных организаций и государственных архивов, административно-хозяйственные, для органов государственного управления, обороны, государственной безопасности, внутренних дел, представительств иностранных государств, финансового назначения, для образования и воспитания, для лечебно-профилактических и санаторно-курортных целей, коммунального хозяйства, культурно-просветительного и зрелищного назначения, религиозного (культового) назначения, для ритуально-похоронного обслуживания, физкультурно-оздоровительного и спортивного назначения, для бытового обслуживания населения, для оказания ветеринарных услуг, для размещения подразделений по чрезвычайным ситуациям, для организации азартных игр, многофункциональных, садовые и дачные домики.


  4. 3.

    Вид работ «на сооружении» устанавливать для пожаров, происшедших:
    • на сооружениях сельскохозяйственного, лесохозяйственного, водохозяйственного, природоохранного, оздоровительного, рекреационного, историко-культурного, культурно-просветительного и зрелищного, физкультурно-оздоровительного и спортивного, научного, религиозного (культового) назначения, для рыбоводства, горнодобывающей, обрабатывающей промышленности, специализированных зданий транспорта, связи, энергетики, коммунального хозяйства, складов, хранилищ, для органов государственного управления, обороны, государственной безопасности, внутренних дел, для размещения подразделений по чрезвычайным ситуациям, гражданской обороны, многофункциональных.
    • на сооружениях неустановленного назначения.


  5. 4.

    Вид работ «на транспорте» устанавливать для пожаров транспортных средств вне зданий и сооружений.
