1. Рекомендации по выезду в соседний гарнизон:

  2. 1.

    По согласованию с дежурной сменой РЦУРЧС на сегодняшний день принято следующее организационное решение по учету выездов в соседние гарнизоны.
    Выезд в соседний гарнизон создает и в дальнейшем ведет диспетчер, на территории которого случилась ЧС при взаимодействии с соседними районами, привлекая и выбирая их технику.
    Соответственно, для исключения дублирования выезд не создается в районе, техника которого уехала в соседний гарнизон.
    На уровне района/области/РЦУРЧС можно при необходимости дополнить (изменить) информацию по данному выезду (например, временные характеристики только по своей технике).
    Напоминаем, что в общей таблице такие выезды отображаются с подсветкой и дополнительным знаком.

  3. 2.

    Для примера: произошла ЧС на территории Ошмянского района. Ошмянский РОЧС привлекает технику из Сморгонского РОЧС. Ошмянский РОЧС СОЗДАЕТ выезд; Сморгонский РОЧС НЕ СОЗДАЕТ выезд, а видит выезд, созданный Ошмянским РОЧС (если установлена настройка отображения подобных выездов в пункте меню Настройки->Другие), и дополняет его информацией по необходимости.
