1. Справочник ПАСП

  2. 1.

    В программном средстве реализован справочник ПАСП (левое меню -> Классификаторы -> Справочник ПАСП).

  3. 2.

    Заходим сюда и для каждого подразделения, доступного в таблице, добавляем адрес(переходя по кнопке соответствующего подразделения).

  4. 3.

    При создании выезда и указании причины "05 занятия" - появляется поле для выбора подразделения, после выбора подразделения адрес выезда автоматически заполняется адресом выбранного подразделения (из справочника ПАСП).
