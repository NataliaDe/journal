• 18 помощь населению

    Поля формы для причины 18 помощь населению
    Обязательные для заполнения Не обязательные для заполнения
    Вид работ Детализированная информация

    Примеры детализированной информации для этой причины вызова:
    1. Извлекали детскую коляску из реки. (Вид работ – извлечение предмета)
    2. Распиловка упавшего дерева на жилой дом. (Вид работ – разборка завалов, уборка аварийных деревьев)
    3. Открывали дверь в квартиру (сломан замок). (Вид работ – открывание дверей)

    ! Виды работ: работы по шорингу - возведение поддерживающих лесов; установка временных опорных стоек; установка креплений траншей; временная опорная стойка.
